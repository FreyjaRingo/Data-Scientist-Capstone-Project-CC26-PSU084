# Orang A - Nutrisi dan Profil Pengguna

Folder ini berisi pekerjaan Data Science bagian Orang A.

- Cleaning dataset nutrisi makanan.
- Membuat food master bersih.

- EDA nutrisi dan profil pengguna.
